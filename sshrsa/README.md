# SSH KEY 加密工具
使用你的ssh私钥来加解密一些小数据，让它们可以安全的放在共享代码中。

代码中的秘钥、token、密码之类的数据，我们都知道应该放在配置文件不要上传到公共的git。

但是一些小项目，不想每次都要写配置文件那么麻烦。就可以用到这个库啦！
